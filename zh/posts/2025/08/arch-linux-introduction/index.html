前言#

欢迎您来到 Arch Linux 的世界。Arch Linux 属于 Linux 操作系统家族，操作系统是沟通应用程序和计算机硬件的桥梁，也就是说，您可以将其安装到计算机中，让它为您驱动硬件，以便在其上运行应用程序。

Arch Linux 支持 x86-64 架构的计算机，包括市面上绝大多数的台式和笔记本电脑。Arch Linux 上的软件包数量约有 8 万个，在 Linux 发行版中数一数二。如果利用虚拟机或兼容层，Windows 或 Android 平台上的大量应用程序，也可以在其上运行。

除了实用性之外，安装并使用 Arch Linux 也有助于您学习计算机知识。由于它遵循简洁的原则，您必须亲自动手安装并配置您所需要的软件。在操作过程中，您将熟悉系统各组件的功能，并获得独立解决计算机常见问题的能力。

此外，Arch Linux 也是很好的玩具。您可以像拼接积木一样，组装您的操作系统，实现您想要的功能。在此过程中，您将感到有所成就，并寻找到志同道合的朋友。

如果您愿意，请和我一起进入 Arch Linux 的世界。预祝您在这里的旅途愉快！

什么是 Linux 操作系统#

Linux 操作系统是一类基于 Linux 内核的开放源代码的操作系统。

Linux 内核由林纳斯·托瓦兹（Linus Torvalds）于 1991 年首次发布。它负责管理计算机硬件资源，如 CPU、内存、存储、网络等，并为软件提供接口。

Linux 内核是自由及开放源代码软件，也就是说只要遵守 GNU 通用公共许可证（GPL），任何个人和机构都可以自由地使用 Linux 内核的所有源代码，也可以自由地修改和再发布。

Linux 操作系统是一个操作系统家族，包括若干 Linux 发行版，如 Debian（及其派生版本 Ubuntu、Linux Mint 等）、Fedora（及其相关版本 Red Hat Enterprise Linux）、openSUSE、Arch Linux、Gentoo Linux 等。一个 Linux 发行版由 Linux 内核、系统工具、图形用户界面（GUI）和应用程序组成。

系统工具主要由理查德·斯托曼（Richard Stallman）于 1983 年发起的 GNU 计划提供，它们也是自由及开放源代码软件。图形用户界面（GUI）主要指的是桌面环境，如 GNOME、KDE 等。Linux 发行版中包含的应用程序一般有文件管理器（如 Dolphin、Thunar）、网页浏览器（如 Firefox、Chromium）、办公软件（如 LibreOffice）、图像处理工具（如 GIMP）等。

什么是 Arch Linux#

Arch Linux 是一个滚动发布的 Linux 发行版。Arch Linux 通过定期更新其包含的各个软件来保持最新。Arch Linux 的设计意图是最小化，用户可以在安装过程中进行配置，这样就可以只添加自己需要的内容。

pacman 是专为 Arch Linux 编写的软件包管理器，用于安装、删除和更新软件包。此外，Arch 用户资源库（AUR）是 Arch Linux 的社区驱动资源库，提供官方资源库未包含的软件包和软件包的替代版本；AUR 软件包可手动下载和构建，或通过 AUR 助手安装。

Arch Linux 有一个名为 ArchWiki 的社区维基（wiki），提供全面的文档。
