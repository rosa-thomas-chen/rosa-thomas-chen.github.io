1 简介#

KataGo 是由 David J. Wu（GitHub：lightvector）所研究并开发的一套围棋软件。

David J. Wu 依照 DeepMind 的 AlphaGo Zero 与 AlphaZero 论文，以及许多在 DeepMind 论文之后的相关研究及原创的研究，大幅改善了训练速度，使 KataGo 成为目前世界上最强的电脑围棋软件之一。

“KataGo”这个名字来源于日语单词“かた”（罗马音：kata，中文：类型）。即使在英语中，武道的形式也被称为“kata”。作者 lightvector 表示，它作为一个通过强化学习永久训练自己并旨在完成自己的形式的人工智能的名称似乎是合理的1。

1.1 特色功能#

相比 AlphaGo，KataGo 的优势在于：

  1. 能够估算目数差，并评估每个交叉点的归属权，而不仅仅是显示胜率。这个功能可以极大地帮助爱好者了解每手棋的作用。
  2. 能够最大化领先优势，在落后时下出胜负手，在领先时减少退让。
  3. 支持自定义贴目和让子棋。
  4. 支持从 7x7 到 19x19 的棋盘尺寸。最大可以支持 29x29 的棋盘（需要使用 bs29 版本）。
  5. 支持多种规则，包括中国规则、日韩规则和中国古代规则等。
  6. 支持多种行棋风格，包括人类各水平等级的棋风，以及历史上各个时期的棋风（需要相应的配置）。
  7. KataGo 是开源软件，您可以自由地获取、使用、改编和分发该软件，也可以参与神经网络的训练。

2 使用方法综述#

2.1 与图形用户界面搭配使用#

对大多数围棋爱好者而言，使用 KataGo 并不是一件容易的事。这是因为 KataGo 没有图形界面，只能在命令窗口中运行，用户向 KataGo 输入文字命令，KataGo 也以文字的形式输出分析结果。普通用户难以掌握 KataGo 的命令，也难以通过一行行文字输出理解 KataGo 传达的意思，因此我们需要另一个程序作为中介，将用户的棋谱转化成命令，将 KataGo 的输出转化成可视化棋盘。这个程序被称为“图形用户界面”。

简单地说，KataGo + 图形用户界面 = 可视化围棋分析工具。笔者推荐的围棋图形用户界面有：

  • Sabaki，
  • Lizzie，
  • Ogatak。

您可以任选一种与 KataGo 搭配使用。

2.2 使用 KaTrain#

您可能会问：用两个程序完成一件事情是不是太复杂了，有没有用一个程序就能完成的方法呢？KaTrain 就是一个 All in One 的工具。它具有 AI 复盘、人机对弈等功能，支持 AI 行棋风格的调整。

对于普通围棋爱好者而言，KaTrain 的功能已经足够使用；对于计算机操作水平较高的围棋爱好者和研究者而言，可以使用“KataGo + 图形用户界面”的方案。本文将以 Windows 系统为例，分别介绍两种使用方法。

2.3 KataGo 的后端版本#

KataGo 有四个后端可供选择：OpenCL (GPU)、CUDA (GPU)、TensorRT (GPU) 和 Eigen (CPU)。

官方网站2上给出的选择建议是：

如果你有性能良好的GPU，要快速让程序运行起来，可以尝试使用 OpenCL。

对于 NVIDIA GPU，TensorRT 通常能提供更优异的性能，但你可能需要从 NVIDIA 官网安装 TensorRT。

如果你没有 GPU，或者你的 GPU 过于老旧/性能不足以支持 OpenCL，且你只想使用纯 CPU 版的 KataGo，可以使用带 AVX2 的 Eigen。

如果你的 CPU 较旧或设备为低端且不支持 AVX2，请使用不带 AVX2 的 Eigen。

CUDA 后端可在安装了 CUDA+CUDNN 的 NVIDIA GPU 上运行，但性能可能不如 TensorRT。

总之，笔者建议

  1. 大多数用户使用 OpenCL 后端；
  2. 拥有 NVIDIA 高端显卡（20 系列及以上）的用户使用 TensorRT 后端；
  3. 以上二者均无法运行的用户，依次尝试带 AVX2 的 Eigen 和不带 AVX2 的 Eigen。

本文以 OpenCL 为例介绍使用方法。

3 安装显卡驱动#

KataGo 的运行需要显卡的支持。如果没有安装最新版本的显卡驱动，可能导致 KataGo 出现错误。

请先确认您的显卡品牌和型号，方法如下。

https://zh.wikihow.com/%E6%9F%A5%E7%9C%8B%E8%AE%A1%E7%AE%97%E6%9C%BA%E7%9A%84%E6%98%BE%E5%8D%A1%E5%9E%8B%E5%8F%B7

如果您使用 NVIDIA 或 AMD 显卡，请打开驱动程序下载页面，选择您的产品型号和操作系统，下载并安装相应的驱动程序。其他品牌显卡的驱动程序也请前往官方网站下载。

https://www.nvidia.cn/geforce/drivers/

https://www.amd.com/zh-hans/support

如果您选择 KataGo 的 TensorRT 后端，还需要前往官方网站安装 CUDA 和 TensorRT。选择 OpenCL 后端的读者无需安装。

https://developer.nvidia.com/cuda-downloads

https://developer.nvidia.com/tensorrt

下载 TensorRT 之前需要注册并填写问卷，较为复杂。笔者没有尝试过，因而无法提供直接支持。这里只提供一些安装教程。

NVIDIA Deep Learning TensorRT Documentation​ docs.nvidia.com/deeplearning/tensorrt/install-guide/index.html

TensorRT安装及使用教程-CSDN博客​ blog.csdn.net/zong596568821xp/article/details/86077553

现在，让我们开始正式安装 KataGo 吧。

4 KaTrain#

KaTrain 是一个 All in One 的围棋复盘和对弈工具，可能是使用 KataGo 的最简单的方式。

KaTrain 的特色功能有

  1. 复盘您的对局并找到目数损失最大的几手。
  2. 与 AI 对弈时，您能立即得到反馈。当您有失误时，可以得到悔棋机会。
  3. 与弱化的、具有不同风格的 AI 对弈。
  4. 自动生成带有 AI 复盘的 SGF 文件。

您可以前往 KaTrain 官方网站及源代码库获取使用帮助。

https://github.com/sanderland/katrain

4.1 安装#

KaTrain 的安装相对容易。请前往 KaTrain 的 GitHub 页面，下载最新的 KaTrain.zip 文件，然后将其解压缩（不是双击打开，而是右键压缩包，点击解压缩）即可。

https://github.com/sanderland/katrain/releases/latest

截至本文最后一次更新，最新版本是 1.14.0，下载链接是

https://github.com/sanderland/katrain/releases/download/v1.14.0/KaTrain.zip

GitHub 在国内访问不够稳定，笔者会将文件上传到国内网盘，以便读者使用。
